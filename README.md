# wazuh-enterprise-pro
Wazuh Enterprise Pro Installer es un script automatizado para desplegar Wazuh Manager, Indexer y Dashboard en Ubuntu 24.04 LTS, optimizado para entornos de prueba y producción.

Este producto esta diseñado para ser usado con fines de validacion de sus funciones, comprobacion de sus operaciones y puesta en funcionamiento. 

Agradecemos su colaboracion y tiempo para evaluar el resultado de un analisis detallado. 

En est primera etapa se busca comprobar su efectividad y resultado.
